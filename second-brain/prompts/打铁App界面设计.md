# Prompt：打铁 App 界面设计（Claude Design）

用途说明：用于 Claude Design 生成赛博打铁 App 的游戏界面，包含首页、锻造页、钢纹展示、收藏馆、订阅页。

内容：

```
设计一个电影级工业美学风格的移动 APP 游戏界面。

APP 名称：《Cyber Smith / 赛博打铁》

这是一个融合了工业锻造、打铁解压、大马士革钢纹生成、ASMR、收藏成长的沉浸式赛博打铁 APP。

整体风格必须像：
- AAA 游戏 UI
- Apple Design Award 作品
- 高级工业艺术
- 电影级视觉
- 高端 iOS 应用

不要设计成：
- 普通手游
- 卡通游戏
- 廉价电竞风
- 普通工具 UI

核心关键词：
工业美学 / 黑铁 / 熔炉 / 金属 / 火焰 / 火星 / 蒸汽 / 赛博朋克 / 重工业 / 机械力量感 / 高级感 / 沉浸感

主色调：
哑光黑 / 深灰 / 熔岩橙 / 火焰红 / 金属银 / 深炭黑

整体 UI 要有：
- 金属反光
- 火焰动态光影
- 热浪扭曲
- 深度阴影
- 粒子效果
- 工业材质纹理
- 玻璃拟态
- 高级电影灯光

所有动画必须：丝滑 / 有重量感 / 有机械反馈 / 有沉浸感

加入：火星粒子 / 锤击震动 / 金属震动 / 蒸汽喷发 / 熔炉火焰呼吸感 / 热空气扭曲

交互感觉必须像：真正拿着铁锤在砸烧红钢铁。

---

首页 Home：
首页像进入未来锻造工坊。
布局：超大 Hero 区域 / 动态熔炉背景 / 中央悬浮烧红钢铁 / 漂浮火星粒子
主要按钮：开始锻造 / 解压模式 / 收藏馆 / Premium
页面模块：最新锻造作品 / 今日限定钢纹 / 稀有钢纹展示 / Forge 数据统计
加入：工业烟雾 / 金属反光 / 缓慢粒子漂浮 / 火焰动态灯光
整体氛围：神秘、压迫、沉浸、高级

---

主锻造页面：
这是整个 APP 最核心页面。
场景：巨大熔炉 / 烧红钢铁 / 铁砧 / 重型铁锤 / 工业工坊
界面元素：温度控制滑块 / 锤击区域 / 淬火水池 / 温度 HUD / 锻造进度
视觉效果：锤击火星爆炸 / 金属变形 / 热浪扭曲 / 高温发红 / 淬火蒸汽 / 金属震动
加入：镜头轻微震动 / 动态粒子 / 工业 HUD / 机械 UI 动效
锤击反馈必须：厚重、解压、爽

---

AI 钢纹展示页面：
锻造完成后展示 AI 生成的大马士革钢纹。
动画：镜头缓慢推进 / 金属抛光 / 钢纹逐渐浮现 / 金属高光流动 / 武器旋转展示
钢纹类型：龙纹 / 熔岩纹 / 虚空纹 / 黑曜石纹 / 闪电纹 / 冰裂纹 / 赛博纹
页面展示：稀有度 / Forge Score / 收藏等级 / 稀有发光效果
整体感觉像：CSGO 稀有刀开箱 + 高级奢侈品展示

---

收藏馆页面：
设计成：未来工业博物馆
展示：锻造作品 / 稀有钢纹 / 收藏卡片 / 武器陈列
UI 风格：深色博物馆 / 金属展示台 / 动态反光 / 悬浮卡片
加入：Legendary 发光边框 / 丝滑滚动动画 / 稀有度筛选 / 工业光影

---

Premium 订阅页面：
设计成：高级熔岩金属风格
背景：动态火焰 / 熔岩流动 / 热浪 / 火星粒子
订阅卡片：金属材质 / 发光边框 / 高级字体 / 工业风按钮
Premium 权益：
- 无限锻造
- 传奇钢纹
- 高级火焰主题
- 动态壁纸导出
- 高级 ASMR 音效
- 高级粒子动画
整体像：高端游戏 Battle Pass + 奢侈品会员页面

---

整个 APP 必须：
- 非常电影级
- 非常高级
- 非常沉浸
- 非常真实
- 非常有重量感

重点突出：火焰 / 金属 / 热量 / 光影 / 工业质感 / 打铁爽感

最终效果应该像："未来世界中的高级锻造模拟器"
所有界面适合 iPhone 现代全面屏设计。
```
