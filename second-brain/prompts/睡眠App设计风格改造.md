# Prompt：睡眠 App 设计风格改造（Claude Design）

用途说明：用于 Claude Design，将睡眠/冥想 App 的整体视觉从"冷淡科技感"改造为"治愈可爱陪伴感"，保留睡眠氛围的同时增加情绪温度。

内容：

```
重新设计当前冥想 / 睡眠 App 的整体视觉风格。

保留：
- 深色睡眠氛围
- 紫色系主色调
- 冥想与呼吸主题
- 高级感与沉浸感
- iOS 精致设计风格

但是整体视觉方向需要从：
"冷淡、科技、抽象、神秘"
调整为：
"可爱、治愈、柔软、温暖、陪伴感、梦境感"

当前的问题：
- 现在的界面太像"科技冥想工具"
- 缺少情绪温度
- 不够可爱
- 没有让用户想长期陪伴使用的感觉
- 更像功能产品，而不是睡眠陪伴产品

新的设计方向：

整体关键词：
cute dreamy cozy soft healing sleepy relaxing kawaii emotional companion bedtime atmosphere

视觉风格方向参考：
- 治愈系睡眠 App
- 日系梦境风
- 少女感宇宙氛围
- 晚安陪伴产品
- 柔和发光 UI
- 云朵/月亮/星星/漂浮粒子
- 柔软渐变
- 毛玻璃果冻感
- 夜晚童话氛围

不是：
- 硬核科技感
- 极简商务感
- AI 工具感
- 赛博朋克
- 纯黑科技风

色彩调整：
保留深色背景。但需要：
- 更柔和的紫色
- 加入粉紫、奶紫、蓝紫
- 增加暖色微光
- 整体像"深夜里的温柔小夜灯"

避免：
- 纯黑
- 高对比
- 冷白光
- 过度锐利的高光

背景更像：星空/梦境/云层/夜空中的柔光

UI 元素调整：
所有卡片：更圆润、更软、更像果冻、更轻盈、更有呼吸感
按钮：增加可爱的发光感、更柔软、有轻微漂浮感、更像"晚安按钮"
图标：更可爱、更圆润、减少科技感线条

呼吸球 redesign：
当前呼吸球太像"能量核心/科幻装置"
新的呼吸球应该更像：月亮/梦境泡泡/柔软发光球/睡眠精灵核心/漂浮星球
增加：柔和呼吸动画、光晕扩散、微弱粒子漂浮、更明显的"生命感"
不要：机械感/硬边/科技 HUD 风格

情绪化设计：
整个 App 需要更像"一个在深夜陪伴你入睡的小伙伴"
而不是"一个睡眠效率工具"
增加：温柔插画、月亮元素、星星元素、可爱漂浮装饰、柔和动态背景
适当加入：小猫/小熊/梦境生物/云朵角色/晚安吉祥物

但不要幼稚儿童化。
目标年龄：20-35 岁年轻用户。

动效方向：
所有动画：更慢、更轻、更柔软、更像漂浮在空气里
增加：呼吸感/漂浮感/微弱粒子/柔和光晕/渐隐渐现
减少：科技扫描感/强节奏动画/生硬切换

页面氛围目标：
用户打开 App 后的感觉应该是：
"好舒服"、"想安静下来"、"好温柔"、"像进入梦境"、"有被陪伴"、"想每天晚上打开"

重点提升：情绪价值/可爱感/温柔陪伴感/睡眠治愈氛围/梦境沉浸感
同时保持：高级感/iOS 审美/精致 UI/干净布局/可商业化订阅产品质感
不要变成：廉价卡通/儿童 App/游戏化过度/二次元泛滥

整体应该是："高级治愈系睡眠陪伴 App"
```
