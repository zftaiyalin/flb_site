# 打铁 App（Cyber Smith / 赛博打铁）

## 产品定位

工业锻造 ASMR + AI 大马士革钢纹生成 + 收藏成长系统。
本质是"赛博 ASMR"——火焰、锤击、金属声、工业氛围。
定位：单机内容生成型 App，用户打铁 → 生成钢纹 → 导出视频 → 发到外部平台（TikTok/抖音/小红书）传播。

## MVP 功能

- 锻造流程：加热 → 锤击 → 淬火 → 生成钢纹
- 极致打击感 + 火花粒子 + 震动反馈
- AI/算法生成大马士革钢纹（参数化生成 + 随机算法）
- 钢纹稀有度展示（像 CS:GO 开箱）
- 本地视频导出（9:16 抖音版 / 1:1 小红书版）
- 收藏馆（陈列历史锻造作品）
- Premium 订阅页面

## 已确定功能

- 完全本地化，不需要服务器（MVP 阶段）
- 无社区系统（依赖用户自行发到外部平台传播）
- 本地视频合成：Flutter + 原生视频导出
- 激励广告（观看广告解锁稀有钢纹/更多锻造次数）
- 一键分享到 TikTok / Instagram / 抖音

## 后续版本功能

- 云端排行榜 + 分享广场（验证有人分享后再做）
- AI 真实生成（当前用参数化伪 AI）
- 更多钢纹风格：金属纹、木纹、石纹、赛博材质

## UI/设计方向

核心关键词：工业美学、黑铁、熔炉、火焰、火星、蒸汽、赛博朋克

**主色调：** 哑光黑 / 深灰 / 熔岩橙 / 火焰红 / 金属银

**主要页面：**
- 首页：动态熔炉背景 + 漂浮火星粒子 + 中央悬浮烧红钢铁
- 主锻造页：温度控制 + 锤击区域 + 淬火水池 + 火星爆炸动效
- AI 钢纹展示：镜头推进 + 金属抛光 + 钢纹逐渐浮现（像高端奢侈品展示）
- 收藏馆：未来工业博物馆风格，Legendary 发光边框
- Premium 页：熔岩金属风格 + 动态火焰背景

## 技术方案

- Flutter 主 App
- Shader + noise + procedural texture 生成钢纹（伪高级感）
- 序列帧火焰 + 粒子贴图 + 镜头震动 + 动态模糊
- 本地视频录制：ReplayKit (iOS) / MediaProjection (Android)
- 订阅：RevenueCat
- 广告：AdMob（主要海外用户）

## 商业模式

- 免费：基础锻造次数 + 基础钢纹
- Premium 解锁：无限锻造 + 传奇钢纹 + 高级火焰主题 + 动态壁纸导出 + 高级 ASMR 音效
- 激励广告：解锁额外锻造次数

## 风险与问题

- 视频生成性能（粒子 + shader + 视频编码会吃 GPU，iPhone 可能发热）
- MVP 阶段用参数化算法而非真 AI，用户是否认可"AI 生成"定位
- 赛道相对冷门，验证需要一定时间
